# 🐍 Snake Game (Single HTML)

HTML 하나에 **HTML + CSS + JavaScript**가 모두 포함된 스네이크(Snake) 게임입니다.  
브라우저에서 바로 실행되며 **화살표 키(← ↑ → ↓)**로 조작합니다.

> 단일 파일 구성이라 배포/공유가 매우 쉽습니다!

---

## 🎮 Demo

- 로컬 실행: `snake.html` 파일을 브라우저에서 열면 즉시 실행됩니다.
- GitHub Pages로 배포할 수도 있습니다 (하단 참고)

---

## ✨ Features

- **완전 단일 HTML 파일**
- **키보드 화살표키로 조작**
- **P 키**: 일시정지 / 재개  
- **R 키**: 즉시 재시작  
- **점수 / 최고점수 기록(LocalStorage)**
- **먹이를 먹으면 속도 증가**
- **벽 충돌 / 자기 몸 충돌 시 게임 오버**
- 반응형 UI / 오버레이 안내

---

## 🕹️ Controls

| Key | Action |
|-----|--------|
| ← ↑ → ↓ | 방향 이동 |
| P | 일시정지 / 재개 |
| R | 재시작 |

※ 반대 방향 즉시 전환은 불가합니다. (예: 오른쪽 이동 중 ← 입력 불가)
